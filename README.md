# How Much Memory Do Linkers Need?

## Development

### Requirements

- Zsh
- GNU time
- CMake >= 3.29 (required by `CMAKE_LINKER_TYPE`)
- Ninja (optional)

### Setup

```console
$ git submodule update --init --recursive --depth=1
```

### Collecting Data

```console
$ ./main.zsh
```

You might want to adjust CMake arguments inside the script.
